# 🎬 Netflix Clone

A Netflix-inspired landing page built with **React** and **Vite** as part of the DevWeekends Fellowship 2026. This project focuses on React fundamentals such as component based architecture, props, state management, dynamic rendering, and responsive UI design.

---

## 🚀 Features

- Built with React + Vite
- Component-based architecture
- Dynamic movie rendering using `.map()`
- Movie data managed through a separate data file
- Reusable `MovieCard` component using props
- Category filtering (All, Action, Comedy, Drama)
- Search functionality to filter movies by title
- Active category highlighting



---

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- CSS3
- React Hooks (`useState`)
- React Icons
